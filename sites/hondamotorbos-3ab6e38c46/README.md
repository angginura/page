# Honda Motor Landing Page

Ini adalah landing page untuk dealer motor Honda, dibuat dengan Vite + Vanilla JS.

## Struktur Project
- `source/`: Source code (HTML, CSS, JS, Data)
- `dist/`: Hasil build SIAP DEPLOY (Static Files)

## Cara Deploy
Kami sudah menyertakan folder `dist/` yang siap diupload.
1. Ambil semua file di dalam folder `dist/`.
2. Upload ke hosting Anda (public_html, Netlify, Vercel, dll).
3. Selesai!

## Pengembangan
Jika ingin mengedit kode:
1. `npm install`
2. `npm run dev` (untuk preview local)
3. `npm run build` (untuk generate ulang folder dist)

## Fitur
- Data motor dari JSON (`source/data/motor.json`)
- Tombol WhatsApp otomatis dengan pesan detil items.
- Responsive design.
- SEO Friendly basic.
